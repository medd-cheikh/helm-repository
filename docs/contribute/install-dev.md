# Contributing

## Install source

```bash
git clone https://github.com/startxfr/helm-repository.git
cd helm-repository
```

## Create a new chart

```bash
./sx-helm mychart create
```

## Test a chart

```bash
./sx-helm mychart test
```

## Package a chart

```bash
./sx-helm mychart package
```

## Release a chart

```bash
./sx-helm mychart package auto
```
