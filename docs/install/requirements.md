# Requirements

## 0. Operating-system

Having a RedHat like (Fedora, CentOS, RHEL, ...) operating system. 

## 1. Git CLI

Install git with `sudo yum install git`

## 2. oc or kubectl CLI

Install openshift client with `sudo yum install openshift-client`

## 3. Helm CLI

Install helm with `sudo yum install helm`
